/*
给定一个整型数组arr，和一个整数num
某个arr中的子数组sub，如果想达标，必须满足：
sub中最大值 – sub中最小值 <= num，
返回arr中达标子数组的数量
*/

/*
    // ps：子数组  类似于字符串的子串，是连续的
    // ps：一个结论： 如果L~R上的max-min <= num   那么L~R的子数组（往里缩）的max-min也<=num  因为子数组的max<=父数组的max  子数组的min>=父数组的min
    // ps：另一个结论：如果L~R上不达标，那么L~R的父数组（往外扩）也不达标

    所以，解题思路：搞两个滑动窗口，一个记录max，一个记录min
    L指向0，R指向1，R往右移动，假设R指向99满足要求，R指向100不满足要求
    那么，可以找到子数组0~0   0~1  0~2   0~3 ... 0~99   以0开头的子数组有100个
    然后L右移动一位，R再尝试向下走，假设R走到了 120 ，可以找到子数组1~1   1~2   1~3   1~4... 1~120
    ......
* */

https://gitee.com/w_y123/mynote_img/raw/master/images/image-20240514034910915.png
